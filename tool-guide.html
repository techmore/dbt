Decision Guide

Which Tool To Use

This page removes guesswork. Pick the tool that matches the time you have and the state you are in. Use the shortest tool that fits the moment, then return to the main routine.

Back to Home Open Worksheets

Decision Ladder

5 to 10 minutes

  • Use the morning or midday routine.
  • Do a body check, paced breathing, and one skill.
  • Success: you used one DBT action on purpose.

10 to 20 minutes

  • Use the diary card or daily practice sheet.
  • Record emotions, triggers, skills, and urges.
  • Success: you captured enough data to review later.

20 to 40 minutes

  • Use the chain analysis or weekly review sheet.
  • Work through one difficult moment carefully.
  • Success: you identified one place to do something differently.

Idle / scrolling urge

  • Open one YouTube skill video or read one workbook section.
  • Choose one topic only: mindfulness, TIPP, DEAR MAN, or opposite action.
  • Success: the screen time moved you toward practice, not away from it.

Tool Matching

If you feel flooded

Use STOP, paced breathing, TIPP, or the crisis plan.

If you cannot name emotions

Use the emotion / trigger / response selector sheet or the diary card.

If something went wrong

Use chain analysis, then write one different step for next time.

If you need to speak up

Use the DEAR request planner or the interpersonal effectiveness section.

If you want to review

Use the weekly review sheet and the progress calendar together.

If you have extra time

Use the workbook PDFs or one focused YouTube skill lesson. Do not turn it into random browsing.

Simple Rule

Start with the smallest tool that matches the moment. If that does not solve it, move one step deeper: daily routine, diary card, chain analysis, weekly review, workbook, then a single reinforcement video.
