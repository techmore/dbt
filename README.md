# DBT Starter Website

This repository contains a static GitHub Pages site with a structured, low-friction DBT practice flow:

- 0-to-1 progression (first 14 days)
- Daily schedule for high emotional load and consistency
- Compact weekly structure
- Skill resources and support links
- Local progress tracker (browser localStorage)

## Local preview

Open `index.html` directly in a browser, or run:

```bash
python3 -m http.server 4173
```

Then visit `http://localhost:4173`.

## GitHub Pages deployment

1. Push this repo to GitHub.
2. In repo settings, enable Pages with branch `main` (or your default branch) and `/ (root)` directory.
3. The site will be served at:
   `https://<your-username>.github.io/<repo-name>/`.

## Notes

This site is educational and does not replace therapy. For urgent crises in the US, call
`988` or local emergency services.
