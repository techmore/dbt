First Move

Start with the next action for today, then use the daily practice sheet or the nighttime scaffold.

What to print first

Daily

Use one sheet per day for the morning and evening routine. This is the core daily practice tool.

Weekly

Use one sheet at the end of the week to review goal completion, streaks, and what to change next week.

Weekly Flow

Monday to Wednesday

Use the diary card every day. Keep the practice small and repeat the same morning and evening blocks. This is your main behavior-tracking tool.

Thursday to Friday

Do one chain analysis if a moment was rough. Use one DEAR request planner if you need to speak up clearly. These are the main repair and communication tools.

Saturday to Sunday

Use the weekly review sheet. Decide whether the next week needs more sleep, more skill practice, or fewer targets. This is the measure-and-adjust step.
