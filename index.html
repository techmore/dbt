DBT: 0 to 1 — Consistent Practice Framework

A practical, low-overhead website based on the structure you can apply daily. Grounded in DBT basics: mindfulness, emotion regulation, distress tolerance, and interpersonal effectiveness.

Guiding Principle

This is a practice system, not a diagnostic or treatment replacement. If you are in crisis, call emergency services or 988 in the US immediately.

What this site gives you

  • A step-by-step starting protocol (hours to weeks)
  • Daily routine you can run in 20–40 minutes total
  • Quick access to DBT skill lists and links
  • Progress check-ins so you can track consistency

What to use it with

  • A coach/therapist, if available
  • Supportive routines for sleep, hydration, and movement
  • Not too much newness at once
  • Small, repeatable habits before deeper work

From 0 to 1: the first 14 days

Day 1–2: Stabilize

Only two habits:

  1. 30-second body scan once every 3 hours
  2. 60-second end-of-day log: one feeling, one trigger, one response

Day 3–5: Build observation

Add one core skill:

  • Mindfulness of current emotions before reacting
  • Say: “I am having the emotion, not being the emotion.”

Day 6–10: Add regulation loop

Add one distress tolerance skill:

  • TIPP (cold water, brief movement, paced breathing)
  • Use on first sign of escalation, then continue the routine

Day 11–14: Build interpersonal capacity

Keep existing habits and add one communication target:

  • Use one DEAR skill sentence during a short real-life interaction
  • Keep interactions brief and specific to one objective

Daily Schedule (about 25–35 min total)

Morning (7–12 min)

  • 5-minute body-first check-in
  • 3 minutes of paced breathing (4-6 count cycle)
  • 10-second values statement: “Today I will prioritize …”

Midday (8–12 min)

  • STOP if activated:
  • Stop · Take a step back · Observe · Proceed mindfully
  • Choose one self-soothing action from distress tolerance list

Evening (8–15 min)

  • Log one chain (trigger → feeling → behavior → result)
  • Name one alternative action for tomorrow
  • Choose tomorrow's one-skill focus

DBT Tool Set

Mindfulness

Core goal: increase awareness before action.

  • Observe, Describe, Participate
  • One thing at a time during one task
  • “This is hard, and I can do it in pieces”

Emotion Regulation

Core goal: reduce emotional volatility.

  • Check current emotions for intensity, duration, context
  • Build one opposite action and do it consistently
  • Target sleep and hydration first

Distress Tolerance

Core goal: survive urges without harm.

  • TIPP protocol at first wave of overwhelm
  • Self-soothing by senses: cold, texture, scent
  • Disaster survival plan for impulsive moments

Weekly Structure (Not Overwhelming)

Week 1

Use only one skill per day and one daily checkpoint. Keep sleep baseline stable.

Week 2

Add emotional tracking in the evening and one interpersonal target on 2 days.

Week 3

Keep previous habits and add one interpersonal communication script on 3 days.

Week 4+

Repeat with one refinement each week: slower pace, clearer boundaries, better recovery speed.
